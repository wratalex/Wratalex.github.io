Mes projets

Cookie Clicker

Coockie Clicker est un projet personnel réalisé en java fx. L'objectif est de cliquer sur le cookie pour augmenter son score. Avec le score on peut acheter des multiplicateurs qui fait augmenter le score plus rapidement lorsque l'on clique.

Demineur

Le démineur est un projet scolaire réalisé en java fx. L'objectif est de localiser des mines cachées dans une grille représentant un champ de mines virtuel, avec pour seule indication le nombre de mines dans les zones adjacentes.

JO Paris

JO Paris est un projet scolaire réalisé en laravel. On peut créer un compte et se connecter. Une fois connecté, on peut créer un sport, uniquement le créateur peut le modifier, supprimer et ajouter une image. Il y a également des comptes administrateurs qui permettent de modifier et supprimer tout les sports.

Traversé du désert

La traversé du désert est projet scolaire en python sans interface graphique. Le but est de traversé le désert sans se faire attrapé. Pour cela on a plusieurs options car il faut faire attention a la soif et la fatigue du chameau. La partie se termine lorsque le joueur a parcouru les 300km ou qu'il a été attrapé.

Flatcraft

Flatcraft est un projet scolaire en java fx. L'objectif est de recréer un minecraft en 2d, avec une base donnée. Il y a les mêmes dimensions que minecraft.

Marathon du web

Le marathon du web est un evenement organisé par l'IUT de Lens ou des étudiants de MMI et Informatique sont en groupe pendant 2 jours. L'objectif était de faire un site ou on peut créer et lire des histoires où on est le héros.
