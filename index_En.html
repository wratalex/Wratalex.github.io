My projects

Cookie Clicker

Cookie Clicker is a personal project made in java fx. The objective is to click on the cookie to increase your score. With the score you can buy multipliers that increase the score faster when you click.

Deminer

The deminer is a school project made in java fx. The goal is to locate mines hidden in a grid representing a virtual minefield, with the only indication of the number of mines in adjacent areas.

JO Paris

JO Paris is a school project made in laravel. You can create an account and log in. Once connected, you can create a sport, only the creator can edit it, delete and add an image. There are also administrator accounts that allow you to edit and delete all sports.

Crossing the desert

Crossing the desert is a python school project with no GUI. The goal is to cross the desert without getting caught. For this we have several options because we must pay attention to the thirst and fatigue of the camel. The game ends when the player has traveled the 300km or has been caught.

Flatcraft

Flatcraft is a school project in java fx. The objective is to recreate a minecraft in 2d, with a base. There are the same dimensions as minecraft.

Web marathon

The web marathon is an event organized by the IUT of Lens where MMI and IT students are in groups for 2 days. The goal was to make a site where you can create and read stories where you are the hero.
